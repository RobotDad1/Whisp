## 🧠 Description

**Whisp** is a modern real-time chat application designed for **secret and private communication**.  
It provides a smooth and minimal interface where users can freely connect, **drop a message or share media**, and have conversations in a secure environment.

👉 **Message me:** [Click here](https://whisp-b5uw.onrender.com)  
(Wait few seconds for server startup)

Whisp makes communication feel effortless and intuitive. Built with a focus on clean UI, multiple themes, and seamless messaging experience.

---

## 🚀 Features

- 🔐 Authentication (OTP + JWT)
- 💬 Real-time messaging with media support
- 📎 Share images, videos, and files instantly
- 🟢 Online/offline presence
- 🎨 Multiple UI themes (Glass, Dark, WhatsApp)
- 🖼️ Profile image upload (Cloudinary)
- 🔁 Password reset via OTP
- 👤 Guest login
- ⚡ Smooth & responsive UI

---

## 🏗️ Architecture

Client (React) ↔ Server (Node/Express) ↔ Database (MongoDB)

- REST APIs for authentication & user management
- WebSockets for real-time messaging
- Cloudinary for media storage
----

## 🛠️ Tech Stack

### 🔹 Frontend
- React (Vite)
- Tailwind CSS
- Zustand (State Management)
### 🔹 Backend
- Node.js
- Express.js
### 🔹 Database
- MongoDB (Mongoose)

### 🔹 Services
- Resend (Email OTP)
- Cloudinary (Image Upload)
